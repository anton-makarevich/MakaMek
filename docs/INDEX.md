# MakaMek Documentation Index

This index is the single entry point for all project documentation. It maps the documentation hierarchy so that AI agents and contributors can locate relevant files without scanning the entire repository.

Navigate to a category index to see available documents with summaries. Load only the documents you need rather than bulk-loading an entire category.
